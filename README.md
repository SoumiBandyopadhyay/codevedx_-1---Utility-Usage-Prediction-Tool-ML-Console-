
# Utility Usage Prediction Tool

## Project Overview
This is a Python-based console application that predicts utility usage using Machine Learning.

The project allows users to:
- Add utility usage data
- View stored data
- Update existing records
- Train a Machine Learning model
- Predict future utility usage

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- CSV File Handling
- Git & GitHub

---

## Project Structure

```text
Utility Usage
│
├── data
│   └── usage_data.csv
│
├── models
│
└── src
    ├── main.py
    ├── data_handler.py
    └── ml_model.py
```

---

## Features
- Menu-driven console system
- CSV file handling
- Add and update data
- Machine Learning prediction
- Exception handling
- Modular code structure

---

## Installation

### Install Required Libraries

```bash
pip install pandas numpy scikit-learn joblib
```

---

## How to Run

### Step 1
Open terminal inside `src` folder.

### Step 2
Run:

```bash
python main.py
```

---

## Sample CSV Data

```csv
month,units_used,bill_amount
1,120,950
2,150,1200
3,180,1400
```

---

## Menu Options

```text
1. Add Usage Data
2. View Data
3. Update Data
4. Train Model
5. Predict Usage
6. Exit
```

---

## Machine Learning Model
The project uses:
- Linear Regression

The model predicts future utility usage based on monthly data.

---

- File handling
- Pandas operations
- Machine Learning basics
